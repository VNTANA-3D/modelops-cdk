# Now

Debugs the value of a variable or a string by printing it to `stdout`.

## Example

```yaml
- module: Now
  props:
    key: __start__
```

## Props:

| Name | Type | Description | Optional | Default |
| ---- | ---- | ----------- | -------- | ------- |
| key | `string` | Key where the timestamp value will be stored on the global state | `false` |  |
